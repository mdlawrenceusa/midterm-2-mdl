Discussion Q'N'A October, 25th, 2016

Responsive Web Site Design - What it is and why its important: A responsive web site design is a collection of html and css which allow the user to fully be responsive and active with the content on the site. It's important because this is what the other users see when they click a link or visit a site online. Responsive Web design is the approach that suggests that design and development should respond to the user behavior and environment based on screen size, platform and orientation. The practice consists of a mix of flexible grids and layouts, images and an intelligent use of CSS media queries.
